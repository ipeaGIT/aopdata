# Merge land use and access data

Merges landuse and access data

## Usage

``` r
aop_merge(aop_landuse, aop_access)
```

## Arguments

- aop_landuse:

  A `data.table` of aop land use data

- aop_access:

  A `data.table` of aop access data

## Value

Returns a `data.table` with landuse and access data
